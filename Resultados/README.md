# Resultados

Este directorio almacena los resultados experimentales y de simulación del proyecto.

## Contenido Esperado

### Simulaciones ANSYS
- Mapas de tensiones (von Mises, principales)
- Mapas de deformaciones
- Gráficos fuerza-desplazamiento
- Curvas de convergencia de malla
- Datos exportados (.csv, .txt)

### Validación Experimental
- Comparativas con datos de literatura
- Tablas de validación
- Gráficos de error y ajuste

### Reducción de Modelo
- Superficies de respuesta generadas
- Coeficientes de regresión
- Métricas de error (RMSE, R²)

### Integración Háptica
- Datos de calibración del dispositivo
- Registros de pruebas de usuario
- Vídeos y capturas del simulador en uso

## Organización

Los resultados se organizan por tipo de análisis o experimento. Los archivos deben tener nombres descriptivos que indiquen:
- Fecha del análisis
- Tipo de resultado
- Parámetros principales utilizados

**Ejemplo:** `2024-01-15_tensiones_fuerza5N_malla02.png`

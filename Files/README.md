# Files

Este directorio contiene los archivos de trabajo y recursos utilizados en el desarrollo del proyecto.

## Contenido Esperado

### Modelos CAD
- Geometrías de la córnea (.step, .iges, .scdoc)
- Archivos de ANSYS Workbench (.wbpj, .mechdat)
- Modelos 3D para Unity/SOFA

### Scripts y Código
- Scripts de Python para post-procesamiento
- Código de integración háptica
- Scripts de reducción de modelo
- Herramientas de análisis de datos

### Configuración
- Archivos de configuración del proyecto
- Parámetros de simulación
- Configuraciones de calibración

### Recursos Auxiliares
- Tablas de propiedades de materiales
- Datos de referencia bibliográfica
- Plantillas y formatos

## Notas

- Los archivos grandes (>100 MB) deben gestionarse con Git LFS o almacenarse externamente
- Mantener una nomenclatura clara y consistente
- Incluir comentarios en scripts y documentación inline cuando sea necesario

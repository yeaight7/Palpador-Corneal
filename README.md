# Palpador Corneal - Simulador Quirúrgico

## Descripción del Proyecto

Este repositorio contiene el desarrollo de un **simulador quirúrgico para entrenamiento en palpado corneal**, una técnica fundamental en cirugía oftalmológica. El proyecto integra simulación biomecánica mediante elementos finitos (FEM), reducción de modelo y una interfaz háptica para proporcionar retroalimentación realista al usuario.

### Objetivos Principales
- Modelar la respuesta biomecánica de la córnea ante palpación/indentación
- Simular la interacción mediante ANSYS (análisis por elementos finitos)
- Reducir el modelo para ejecución en tiempo real
- Integrar con dispositivos hápticos para entrenamiento realista

## Estructura del Repositorio

```
Palpador-Corneal/
├── Informes por Partes/    # Documentación técnica por secciones
├── Resultados/             # Resultados de simulaciones y análisis
├── Files/                  # Archivos de trabajo y recursos
├── Vídeos/                 # Material audiovisual y demostraciones
└── Memoria_final.tex       # Memoria final del proyecto (LaTeX)
```

### Directorios

- **`Informes por Partes/`**: Contiene los informes parciales de cada componente del proyecto (ANSYS, Unity, interfaces hápticas, etc.). Cada sección tiene su propia documentación detallada.

- **`Resultados/`**: Almacena los resultados de las simulaciones, incluyendo mapas de tensiones, deformaciones, gráficos fuerza-desplazamiento y datos de validación.

- **`Files/`**: Archivos de trabajo, modelos CAD, archivos de configuración y otros recursos necesarios para el proyecto.

- **`Vídeos/`**: Vídeos demostrativos del simulador en funcionamiento, capturas de pantalla y material para presentaciones.

## Tecnologías Utilizadas

- **ANSYS Workbench/Mechanical**: Simulación FEM de la córnea
- **LaTeX**: Documentación y memoria del proyecto
- **Dispositivos hápticos**: Interfaz para retroalimentación táctil
- **Reducción de modelo**: Para simulación en tiempo real

# Vídeos

Este directorio contiene el material audiovisual del proyecto para demostraciones, presentaciones y documentación.

## Contenido Esperado

### Demostraciones del Simulador
- Vídeos del simulador en funcionamiento
- Interacción con el dispositivo háptico
- Casos de uso y escenarios de entrenamiento

### Material para Presentación
- Clips para la exposición final (máx. 40 min)
- Capturas de pantalla del simulador
- Animaciones de resultados de ANSYS
- Visualizaciones de campos de tensiones/deformaciones

### Tutoriales y Documentación
- Guías de uso del simulador
- Procedimientos de calibración
- Demostraciones técnicas de componentes específicos

## Formato y Calidad

- Preferiblemente formatos estándar: MP4, AVI, MOV
- Resolución mínima recomendada: 1080p para demostraciones
- Incluir subtítulos o narración cuando sea apropiado
- Nombrar archivos descriptivamente: `demo_palpacion_fuerza_variable.mp4`

## Notas

- Los archivos de vídeo pueden ser grandes; considerar almacenamiento externo (Google Drive, OneDrive) si superan el límite de GitHub
- Mantener copias de respaldo de material importante
- Incluir vídeos de mayor calidad para la presentación final
